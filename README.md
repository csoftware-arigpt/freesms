# FreeSMS
Free sms using TextBelt and proxy
## Required
Internet connection and Python 3.x

## How to install
Clone repo

```
$ git clone https://github.com/csoftware-arigpt/freesms.git
$ cd freesms
```
Install dependencies: 
` $ pip3 install requests `

## How to use
Just open file

```
$ python3 main.py
```

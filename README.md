# FreeSMS
Free sms using TextBelt and proxy
## Required
Internet connection and Python 3.x

## How to install
Just clone repo

```
$ git clone https://github.com/csoftware-arigpt/freesms.git
$ cd freesms
```

## How to use
Just open file

```
$ python3 main.py
```
